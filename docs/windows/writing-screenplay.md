# 写作剧本窗口工作指南

## 窗口定位

本窗口研究 CodeX 在创意写作、剧本、人物、对白、结构、风格控制和审稿中的协作方式。

## 适合处理的问题

- 如何生成和评估故事设定。
- 如何搭建大纲、场景和人物关系。
- 如何打磨对白和节奏。
- 如何让 CodeX 做批判性审稿。
- 如何维护长期设定一致性。

## 不适合处理的问题

- 资料整理，转到 `资料整理`。
- 学习计划，转到 `自我学习`。
- 通用提示词研究，转到 `提示描述`。

## 标准工作流程

1. 明确创作阶段：设定 / 大纲 / 场景 / 对白 / 修改 / 审稿。
2. 提供已有文本、风格要求和不可破坏的设定。
3. 要求 CodeX 先指出结构问题，再给修改方案。
4. 对输出进行人物声音、逻辑和节奏检查。
5. 把有效流程沉淀为创作模板。

## 产出物

- 剧本开发模板。
- 人物档案模板。
- 场景拆解模板。
- 审稿 checklist。
