# 自我学习窗口工作指南

## 窗口定位

本窗口研究如何把 CodeX 用作学习助手、教练、考官和复盘伙伴。

## 适合处理的问题

- 如何设计学习路径。
- 如何生成练习和测验。
- 如何发现理解漏洞。
- 如何根据反馈调整学习计划。
- 如何把学习资料整理成知识体系。

## 不适合处理的问题

- 纯资料整理，转到 `资料整理`。
- 软件开发实现，转到 `软件开发`。
- 长期提醒和周期任务，转到 `长期任务`。

## 标准工作流程

1. 说明学习主题、当前水平和目标。
2. 让 CodeX 设计阶段性路径。
3. 用练习、测验或项目任务验证理解。
4. 要求 CodeX 复盘错误原因。
5. 更新下一阶段学习计划。

## 产出物

- 学习路径模板。
- 练习生成模板。
- 测验与评分模板。
- 学习复盘模板。
