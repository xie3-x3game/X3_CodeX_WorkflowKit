# 工具插件窗口工作指南

## 窗口定位

本窗口研究 CodeX 的工具系统，包括 shell、文件编辑、浏览器、GitHub、自动化、Skills、插件和外部连接能力。

## 适合处理的问题

- 某个工具适合什么任务。
- 工具不可用时如何降级。
- Skills 什么时候应该触发。
- 插件和普通工具的边界。
- 新项目需要预设哪些工具流程。

## 不适合处理的问题

- 具体业务任务。
- 单纯提示词优化。
- 深入软件实现，转到 `软件开发`。

## 标准工作流程

1. 明确任务需要哪类外部能力。
2. 记录可用工具和限制。
3. 设计最小工具调用路径。
4. 验证工具结果是否可靠。
5. 把稳定用法写成工具指南。

## 产出物

- 工具能力矩阵。
- Skill 使用规则。
- 插件使用规则。
- 工具降级方案。

## 回填总控

稳定规则回填到 `docs/topics/04-tools-skills-plugins.md`。
