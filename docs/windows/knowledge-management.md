# 资料整理窗口工作指南

## 窗口定位

本窗口研究 CodeX 在资料整理、摘要、结构化输出、索引、标签和知识库维护中的用法。

## 适合处理的问题

- 如何整理长文、碎片笔记、网页资料和文档。
- 如何抽取结构、观点、证据和行动项。
- 如何避免丢失关键细节。
- 如何把资料转成可复用知识库。

## 不适合处理的问题

- 学习路径设计，转到 `自我学习`。
- 创作扩写，转到 `写作剧本`。
- 工具能力研究，转到 `工具插件`。

## 标准工作流程

1. 明确资料类型和整理目标。
2. 先建立目录、标签或信息架构。
3. 分批处理资料，保留来源和证据。
4. 输出摘要、索引、术语表或行动项。
5. 把可复用结构沉淀为模板。

## 产出物

- 资料整理模板。
- 摘要模板。
- 知识库目录规范。
- 来源追踪规则。

## 回填总控

稳定资料整理方法回填到 `docs/topics/09-knowledge-management.md`。
