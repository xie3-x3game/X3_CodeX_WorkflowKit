# 多项代理窗口工作指南

## 窗口定位

本窗口研究多窗口、多代理、并行任务拆分和结果合并。重点是协作结构，不是某个具体专项任务本身。

## 适合处理的问题

- 哪些任务应该拆到多个窗口。
- 哪些任务适合交给子代理并行处理。
- 如何避免多个窗口重复研究。
- 如何合并多个窗口的结论。

## 不适合处理的问题

- 单个窗口内的具体研究。
- 具体代码实现。
- 具体剧本或资料整理任务。

## 标准工作流程

1. 判断任务是否有明确可拆分边界。
2. 定义每个窗口或代理的职责。
3. 规定产出格式。
4. 合并结果时检查冲突、重复和缺口。
5. 把可复用拆分方式写成模板。

## 产出物

- 多窗口分工模板。
- 子代理任务描述模板。
- 结果合并 checklist。
- 冲突处理规则。
