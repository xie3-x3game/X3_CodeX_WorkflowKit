# Rule Update Inbox

This file is the local inbox for WorkflowKit rule updates.

Project total-control reviews this file and decides whether each update should be applied, deferred, rejected, or superseded in this project.

Do not paste raw project materials here. Keep only rule-update decisions and short local notes.

## Status Values

- `Pending`: received but not reviewed.
- `Applied`: accepted and distributed locally.
- `Deferred`: valid, but not useful right now.
- `Rejected`: not suitable for this project.
- `Superseded`: replaced by a newer update.
